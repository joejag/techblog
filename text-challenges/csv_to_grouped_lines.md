Using the editor/tools you currently use on your project. 
Convert this list of strings into the expected CSV output. 

For example.

TNJQCMDP,MBXHDDJB
{% endhighlight %}

{% highlight csv %}
1,DDBQRGGR
1,TVJWJMZJ
2,TNJQCMDP
2,MBXHDDJB
{% endhighlight %}
